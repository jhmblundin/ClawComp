ClawComp Application Page Flow:

Basic info:

1. Name
2. University
3. Projected Graduating Yea
4. University Email
5. Phone Number
6. Major

University email has to be authenticated before moving onto this next portion of the application below:

DISPLAY THIS NOTE:

{cool information icon}: Note that you are expected to apply to this event as a team of 2-3. Has one of your teamates/friends already registered a team? If so, please find it below. If not, REGISTER YOUR TEAM HERE
NOTE END

(Clicking REGISTER YOUR TEAM HERE brings up a modal where the user can create a new team and join it). No duplicate team names are allowed. Once 3 people join a team, it is maxed out. If a user registers a team, autofill their response in the next dropdown (5). The User that creates a team specifies the team size is and the email(s) that they are expecting to fill the remaining slot(s). When they enter in these emails, the people recieve email invites to join their team on the website.

5. Dropdown menu to select team. When a user selects a team, the backend verifies that their email they entered into the form matches the slot that is reserved for them. If the emails don't match, then an error is thrown saying "You do not have a spot reserved on this team under the email {email_they_entered}

6. Link Drop: drop some links to cool things you've built! This could be anything. (ie: personal website, web app you deployed, online art portfolio, online videogame you vibecoded).

7. What are you currently working on? If you don't have a project you are working on, what app/product/service/research would you build right now if you had all the time in the world?

8. Is the team you signed up with currently working on the same project as you?

9. Name 3 things you would be eager to get OpenClaw to do for you. For inspiration, visit our home page, where you can watch videos on OpenClaw customizations or head to the News/stories tab to hear what crazy things OpenClaw is capapble of.
